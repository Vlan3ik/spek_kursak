## БД "Судебные приставы"

**Таблицы:**

1. **Сотрудники (Код сотрудника, ФИО, Возраст, Пол, Адрес, Телефон, Паспортные данные, Код должности, Код отдела) [10 записей].**
2. **Должности (Код должности, Наименование должности, Оклад, Обязанности, Требования) [5 записей].**
3. **Отделы (Код отдела, Наименование отдела, Описание) [3 записей].**
4. **Исполнительные производства (Код ИП, Номер ИП, Дата возбуждения, Код должника, Код взыскателя, Код сотрудника, Сумма долга, Основание, Статус) [15 записей].**
5. **Должники (Код должника, ФИО, Дата рождения, Адрес, Паспортные данные, ИНН, Описание) [15 записей].**
6. **Взыскатели (Код взыскателя, ФИО, Адрес, Телефон, Описание) [10 записей].**
7. **Арестованное имущество (Код имущества, Код ИП, Наименование, Описание, Стоимость, Статус) [10 записей].**
8. **Суды (Код суда, Наименование, Адрес, Телефон) [5 записей].**
9. **Документы (Код документа, Код ИП, Код суда, Наименование, Дата, Описание) [20 записей].**

**Запросы:**

1. **Отдел кадров (Связывает таблицы «Сотрудники», «Должности» и «Отделы» по полям «Код должности» и «Код отдела»).**
2. **Список исполнительных производств (Связывает таблицы «Исполнительные производства», «Должники», «Взыскатели» и «Сотрудники» по полям «Код должника», «Код взыскателя» и «Код сотрудника»).**
3. **Арестованное имущество по ИП (Связывает таблицы «Арестованное имущество» и «Исполнительные производства» по полю «Код ИП»).**
4. **Документы по ИП (Связывает таблицы «Документы», «Исполнительные производства» и «Суды» по полям «Код ИП» и «Код суда»).**

**Фильтры:**

1. **Фильтры для отображения сотрудников отдельных должностей (На основе запроса «Отдел кадров»).**
2. **Фильтры для отображения сотрудников отдельных отделов (На основе запроса «Отдел кадров»).**
3. **Фильтры для отображения исполнительных производств по статусу (На основе запроса «Список исполнительных производств»).**
4. **Фильтры для отображения исполнительных производств по должнику (На основе запроса «Список исполнительных производств»).**
5. **Фильтры для отображения исполнительных производств по взыскателю (На основе запроса «Список исполнительных производств»).**
6. **Фильтры для отображения исполнительных производств по сотруднику (На основе запроса «Список исполнительных производств»).**
7. **Фильтры для отображения арестованного имущества по статусу (На основе запроса «Арестованное имущество по ИП»).**
8. **Фильтры для отображения документов по суду (На основе запроса «Документы по ИП»).**

**Дополнительные задания:**

1. **Создайте представление, отображающее общую сумму долга по всем исполнительным производствам.**
2. **Создайте процедуру, которая будет автоматически обновлять статус исполнительного производства, если сумма долга равна нулю.**
3. **Создайте триггер, который будет запрещать удаление записей из таблицы «Исполнительные производства», если статус производства не равен «Завершено».**

**Примечания:**

* В таблице «Исполнительные производства» поле «Статус» может принимать значения: «В исполнении», «Приостановлено», «Завершено».
* В таблице «Арестованное имущество» поле «Статус» может принимать значения: «Арестовано», «Реализовано», «Снято с ареста».
